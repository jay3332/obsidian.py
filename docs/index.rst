obsidian.py
-----------
A wrapper around Obsidian's REST and Websocket API.

**Features**

- Fully asynchronous
- Object oriented
- Playlist support
- Advanced audio filter support
- Querying tracks from many sources
- Spotify support
- Preset classes
- Fully typed
